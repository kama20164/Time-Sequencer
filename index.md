# Time Sequencer プライバシーポリシー

最終更新日: 2026年5月23日

## 概要

Time Sequencer（以下「本アプリ」）は、ユーザーのプライバシーを尊重します。本ポリシーでは、本アプリおよび本アプリに組み込まれた第三者サービスが、どのような情報を扱うかを説明します。

## 収集する情報（開発者による収集）

本アプリは、**利用者を識別するためのアカウント登録を行いません**。

本アプリが開発者のサーバーへ送信する情報はありません。開発者が、利用者個人を特定する目的で利用状況を追跡することは行いません。

本アプリが端末内で扱う主な情報は次のとおりです。

- **ユーザーが本アプリ内で作成したタイマー（ブロック構成・名称・設定値など）**  
  これらは**お使いの端末内**（アプリの Documents フォルダ内の JSON ファイルなど）に保存されます。
- **アプリの設定（最後に開いたタイマー、サンプル投入の有無など）**  
  端末内の UserDefaults に保存されます。

上記は、本アプリの機能を提供するために端末内で処理され、開発者が運営するサーバーへ送信されることはありません（本アプリの通常の動作の範囲）。

## 広告について（Google AdMob）

本アプリは、無料で提供するため、**Google LLC が提供する広告配信サービス「Google AdMob」**（以下「AdMob」）を利用し、画面下部などにバナー広告を表示することがあります。

広告の表示・計測・不正防止のため、AdMob およびそのパートナー（広告ネットワーク等）が、お使いの端末から次のような情報を**自動的に収集・利用する場合があります**。

- デバイスに関する情報（機種、OS バージョン、画面サイズ、言語設定など）
- 広告識別子（IDFA 等）および、それに代わる識別子
- IP アドレス（おおよその地域の推定に利用されることがあります）
- 広告の表示・クリック・視聴に関する情報
- アプリのクラッシュやパフォーマンスに関する診断情報

これらの情報の取り扱いは、**Google および各パートナーのプライバシーポリシー**に従います。開発者は、AdMob 経由で収集された情報を、利用者個人を直接特定する目的で利用することはありません。

## お使いの端末での設定（広告の管理）

iOS の設定から、広告に関する選択を行うことができます。

- **設定 → プライバシーとセキュリティ → トラッキング**  
  （アプリがトラッキング許可を求めた場合の許可・拒否）
- **設定 → プライバシーとセキュリティ → Apple 広告**  
  （パーソナライズド広告の制限など）
- **設定 → プライバシーとセキュリティ → 分析と改善**  
  （分析データの共有の制限など）

本アプリのバージョンによっては、提出・審査の都合により一時的に広告表示を無効化している場合があります。広告が表示されるバージョンでは、上記のとおり AdMob が利用されます。

## 第三者サービス・解析ツール

- **広告**: Google AdMob（上記）
- **アクセス解析 SDK**: 本アプリでは、開発者独自の利用状況解析 SDK は組み込んでいません（AdMob が収集する情報は除く）。

## お子様のプライバシー

本アプリは、13 歳未満のお子様を対象として設計したものではありません。13 歳未満の方から意図的に個人情報を収集することはありません。

## 本ポリシーの変更

法令の改正、本アプリの機能追加（広告の有無の変更を含む）、第三者サービスの変更等に伴い、本ポリシーを改定することがあります。重要な変更がある場合は、本ページの「最終更新日」を更新し、必要に応じて App Store の商品ページ等でお知らせします。
