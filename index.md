# Time Sequencer プライバシーポリシー

最終更新日: 2026年5月18日

## 概要
Time Sequencer（以下「本アプリ」）は、ユーザーのプライバシーを尊重します。本ポリシーでは、本アプリがどのような情報を扱うかを説明します。

## 収集する情報
本アプリは、**利用者を識別するためのアカウント登録を行いません**。

本アプリが扱う主な情報は次のとおりです。
- **ユーザーが本アプリ内で作成したタイマー（ブロック構成・名称・設定値など）**  
  これらは**お使いの端末内**に保存され、開発者が運営するサーバーへ送信されることはありません（本アプリの通常の動作の範囲）。

## 第三者サービス・解析ツール
本アプリのversion1.0において、**広告 SDK やアクセス解析 SDK を組み込んでおらず**、開発者が利用状況を個人として追跡することは行いません。

## お問い合わせ
本ポリシーに関するお問い合わせは、App Store の商品ページに記載のサポート窓口（サポート URL）からご連絡ください。
